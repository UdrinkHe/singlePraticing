# 简历网站制作
## 项目简介
### 技术使用
1.html+css+js  
2.Vue  
3.打包工具  
4.部署工具  
github page  
### 项目功能介绍
在线编辑简历信息，预览pdf效果。导出编辑信息为pdf形式
### 项目功能要点
1.划分模块，每个模块有不同的输入方式  
2.将输入好的信息导出为pdf（需要有在线预览）
## 项目实现
### 导出pdf的方法  
1.html2canvas+jspdf库  
参考网站：  
html2canvas:https://www.html2canvas.cn/
